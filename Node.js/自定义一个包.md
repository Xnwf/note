### 包

1. 什么是包

   包就是一个文件夹,用来管理模块和模块之间的各种关系

2. 包使用

   `npm install xxx`

   `const xxx = require(xxx);`

3. 自定义包的流程

   1. 新建一个包文件夹
   2. 在包根目录创建`package.json`文件
   3. 在包中编写业务逻辑代码
   4. 上传到`npm`官网
